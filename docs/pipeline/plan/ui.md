# TulennTv — Diseño UI/UX

## Pantallas

### 1. Zapper (Pantalla principal modo TV)
- Video a pantalla completa con overlay
- Nombre del canal (serie) + episodio actual en esquina
- Al cambiar canal: transición tipo TV (barrido/corte)
- Mini-guía inferior al cambiar canal (próximos 3-4 canales)
- Controles: ← → (cambiar canal), ↑ (guía), ↓ (info), ESC (menú)

### 2. Guía de Canales (EPG)
- Grid: filas = canales, columnas = tiempo
- Favoritos marcados con estrella
- Al seleccionar: vista previa + botón "Sintonizar"
- Búsqueda por nombre de serie

### 3. Biblioteca
- Vista en grid con pósters
- Filtros: Todo / Anime / Series / Películas / Favoritos
- Búsqueda
- Al hacer clic: ver episodios de la serie

### 4. Reproductor
- Controles: play/pause, volumen, barra de progreso
- Fullscreen
- Al finalizar: sugerencia del siguiente episodio o cambiar canal

### 5. Perfiles
- Selector al inicio
- Cada perfil con su avatar (emoji)
- Botón "Agregar perfil"

## Tema
- Oscuro (tipo TV/modo cine)
- Colores: fondo #0a0a0a, acento #e50914 (rojo Netflix-like)
- Fuente: sans-serif, tipografía grande y legible (tipo TV)

## Responsive
- PC: ventana redimensionable, controles con mouse/teclado
- TV: optimizado para pantallas grandes, navegación con control remoto
