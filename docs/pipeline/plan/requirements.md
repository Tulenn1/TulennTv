# TulennTv — Requerimientos

## Funcionales

1. **Escaneo de biblioteca**
   - Escaneo automático de carpeta raíz configurable
   - Detección de archivos de video (.mp4, .mkv, .avi, .mov, .webm)
   - Agregado manual de series/carpetas
   - Identificación de series por nombre de carpeta
   - Ordenamiento de episodios por número/temporada

2. **Zapping TV**
   - Cada serie = un "canal"
   - Navegación con flechas (← → cambiar canal)
   - Reproducción continua (al terminar un episodio, siguiente)
   - Preview del canal actual

3. **Guía de canales (EPG)**
   - Grid visual con todos los canales
   - Qué se está reproduciendo ahora en cada canal
   - Próximos episodios

4. **Gestión de favoritos**
   - Marcar/desmarcar series como favoritas
   - Los favoritos aparecen primero en la guía

5. **Historial**
   - Registro de episodios vistos
   - Continuar desde donde se dejó

6. **Perfiles**
   - Múltiples perfiles de usuario
   - Cada perfil con sus favoritos, historial y progreso

7. **Reproductor**
   - Reproducción de video embebido
   - Controles básicos (play/pause, volumen, fullscreen)
   - Avance/retroceso

## No Funcionales

1. **Rendimiento**: La app debe ser ligera, arranque rápido
2. **Offline**: 100% local, sin conexión a internet requerida
3. **Portabilidad**: Misma base de código para PC y TV
4. **Idioma**: UI en español
